Standings CSV Format

season team_code team_name conference gp wins losses win_pct conf_gp conf_wins conf_losses conf_pct sos clinched playoff_wins playoff_losses result 2026 AZ Arizona Rattlers Western 13 8 5 0.615 10 7 3 0.700 0.481 X 0 0 2009 TBL Tampa Bay Storm — 16 14 2 0.875 0 0 0 0 0 3 0 Won United Bowl (RiverCity) 71-62

Include a season column with the year. You can mix multiple years in one file — uploading only replaces the years present in your file; other years stay untouched. If omitted, season defaults to 2026. playoff_wins, playoff_losses, and result are optional and show on historical seasons that don't have conference/SOS data.
